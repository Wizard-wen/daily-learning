
npm全局安装的路径安装在

/usr/local/lib/node_modules/

- npx

软连接 /usr/local/bin/npx 

原路径 /usr/local/lib/node_modules/npm/bin/npx-cli.js

- npm

软连接 /usr/local/bin/npm 

原路径 /usr/local/lib/node_modules/npm/bin/npm-cli.js

- node

/usr/local/bin/node 可执行文件


Mac系统的环境变量，加载顺序为：

/etc/profile 

/etc/paths 

~/.bash_profile 

~/.bash_login 

~/.profile 

~/.bashrc
